# Proposed problems for the laboratory

## Task 1

1. Check if the inverse of a triangular matrix is ​​also triangular. Check if the sum of two triangular matrices is also triangular.
2. Show that any matrix can be written uniquely as the sum of a symmetric matrix and an antisymmetric matrix.
3. Check if a matrix is ​​Hermitian. If so, check if the inverse is also Hermitian.

## Task 2

4. Solve a linear system of 4 equations with 4 unknowns using Gauss's method.
5. Calculate the rank of a matrix using Gauss's method. Calculate the inverse of a matrix using Gauss's method.

## Task 3

6. Given (Zn ,·) the residue group modulo n with respect to multiplication, calculate the order of an element in this group, then check if it is idempotent.
7. Calculate the order of a matrix in the group (M2(R), ∙). Then check if it is idempotent.

## Task 4

8. (from Code Theory) Given a code in (Fq)^n, display the code parameters (n, M, d, s, t,ρ).
9. Check whether a given code in (Fq)^n is linear or not.
10. Display all the words of a code whose generating matrix is ​​known.
11. The control matrix of a code and a set of words are read. Display those that are words of the code.

## Task 5

12. Calculate the eigenvalues ​​associated with a matrix of order 2 or 3, displaying the associated characteristic polynomial.
13. Determine the Galois group associated with a polynomial of degree 3 or 4.

## Task 6

14. Using Euclid's algorithm, calculate the gcd(d) for two given natural numbers (a,b), write a linear combination of the three (a,b,d), display the continued fraction associated with a/b.
