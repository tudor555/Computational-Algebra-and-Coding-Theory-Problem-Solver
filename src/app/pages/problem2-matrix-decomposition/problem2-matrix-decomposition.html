Rezultat: A = S + K

Decompoziția se obține folosind formulele standard din algebra liniară:

  • Componenta simetrică S se calculează ca: S = (A + Aᵀ) / 2
  • Componenta antisimetrică K se calculează ca: K = (A − Aᵀ) / 2

Observație: orice matrice pătratică se poate scrie în mod unic ca sumă între o matrice simetrică și una antisimetrică, deoarece: A = S + K.
